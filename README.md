# 🚀 Evalora AI — Resume Intelligence & Hiring Assistant

Evalora AI is an AI-powered hiring assistant that analyzes resumes, evaluates candidate-job fit, generates interview questions, and provides explainable hiring recommendations.

It transforms raw resumes into structured hiring intelligence using a combination of rule-based logic and AI-driven analysis.

---

## 🎯 Project Objective

To automate and enhance the resume screening process by:

- Reducing manual effort in hiring
- Providing consistent and explainable evaluations
- Generating intelligent interview questions
- Identifying skill gaps and candidate strengths

---

## ⚙️ Features

### 🔍 Resume Analysis
- Supports **PDF, DOCX, TXT**
- Extracts and processes resume content
- Normalizes unstructured data into structured insights

### 📊 Intelligent Scoring System
- Overall candidate score
- Detailed score breakdown:
  - Skills
  - Projects
  - Experience
  - Education
  - Job Match
  - Communication

### 🧠 AI-Based Evaluation
- Candidate summary generation
- Strengths & weaknesses detection
- Explainable hiring recommendation

### 🎯 Skill Match Analysis
- Match score with job role
- Keyword coverage analysis
- Matched vs missing skills

### 📄 Resume Quality Metrics
- ATS score
- Confidence score
- Formatting & clarity evaluation

### 💬 AI Interview Assistance
- Adaptive difficulty questions:
  - Easy / Medium / Hard
- AI-generated follow-up questions

### 📌 Recruiter Dashboard
- Hiring recommendation
- Decision reasoning
- Stored reports (SQLite database)
